# FilcoKeyMap
 A AutoHotkey script which maps navigation keys to the main key area, just like Filco MINILA Air keyboard
## Key Maps
* **CapsLock + s :    ←**  
* **CapsLock + f :    →**  
* **CapsLock + e :    ↑**  
* **CapsLock + d :    ↓**  

* **CapsLock + j :    Insert**
* **CapsLock + m :    Delete**
* **CapsLock + k :    Home**
* **CapsLock + , :    End**
* **CapsLock + l :    PageUp**
* **CapsLock + . :    PageDown**
* **CapsLock + ; :    Backspace**

* **CapsLock + i :    PrintScreen**
* **CapsLock + o :    ScrollLock**

* **Double click CapsLock to trigger 'CapsLock'**
